3,python循环语句

循环是计算机计算解决问题的最常见的思想，计算机里常见的循环有while和for循环。

3.1 while循环

这是一个循环语句，会一直循环下去，是死循环，条件为真时会一直循环。我们不应该把常量作为while的判断语句，因为常量不会变，条件的判断结果也不会变。

while条件判断为假时会打印出 “EOF”。
不过循环一般用for循环的多，不过有一种情况下while非常合适，就是递归。
